# Assets Directory

Place your image assets in the following directories:

## Images (`assets/images/`)
General images and graphics.

## Illustrations (`assets/illustrations/`)

Place the following illustration files here:

### Hero Section
- `handshake.png` - Main handshake illustration for hero section

### Arbeitnehmer (Job Seeker) Steps
- `step1_arbeitnehmer.png` - Step 1 illustration
- `step2_arbeitnehmer.png` - Step 2 illustration  
- `step3_arbeitnehmer.png` - Step 3 illustration

### Arbeitgeber (Employer) Steps
- `step1_arbeitgeber.png` - Step 1 illustration
- `step2_arbeitgeber.png` - Step 2 illustration
- `step3_arbeitgeber.png` - Step 3 illustration

### Temporärbüro (Temporary Agency) Steps
- `step1_temporarburo.png` - Step 1 illustration
- `step2_temporarburo.png` - Step 2 illustration
- `step3_temporarburo.png` - Step 3 illustration

## Note
If images are not found, the app will display placeholder icons. Make sure all images are properly named and placed in the correct directories.
